# Architecture Decision Records (ADR)

本目录用于沉淀 `router-layer` 的架构决策，采用轻量 ADR 流程。

## 使用规则

- 新决策使用连续编号：`0006-...md`、`0007-...md`。
- 已接受的 ADR **不修改结论**；若要变更，新增 superseding ADR。
- 每个 ADR 必须包含：上下文、决策、后果、备选方案。
- `ARCHITECTURE.md` 维护 ADR 索引与状态（Accepted / Superseded / Proposed）。
